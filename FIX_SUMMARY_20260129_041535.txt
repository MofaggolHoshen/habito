--------
React Native error: "Text strings must be rendered within a <Text> component"

ROOT CAUSES:
------------
1. Boolean values in style arrays (e.g., condition && styles.X)
   - When condition is false, React Native tries to render 'false'
   
2. Numbers rendered directly without String() conversion
   - React Native strict mode requires explicit string conversion

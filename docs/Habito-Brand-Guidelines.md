- **Age**: 25-40 years old
- **Profile**: Young professionals, entrepreneurs, students

  - Losing track of daily progress
  - Generic todo apps don't reflect on their day
- **Motivation**: Personal growth, achieving goals, living with purpose

#### Secondary Persona: "The Habit Builder"

- **Age**: 30-50 years old
- **Profile**: Parents, career professionals, wellness enthusiasts
- **Goals**: Establish healthy routines, maintain consistency
- **Pain Points**:
  - Too many tools, none feel right
  - Need visual feedback on progress
  - Want to track "how I'm doing" not just "what I did"
- **Motivation**: Better health, more productive days, sense of accomplishment

## Typography

### Primary Typeface

**-apple-system / Roboto** (System fonts)

**Rationale**:

- Native feel on each platform
- Excellent readability
- No additional font files
- Performance optimization

### Type Scale

#### Headings

```
H1 - Display/Hero
Font: 32px, Bold (700)
Line Height: 40px
Use: Landing pages, hero sections

H2 - Section Title
Font: 24px, Bold (700)
Line Height: 32px
Use: Screen titles, modal headers

H3 - Subsection
Font: 20px, Bold (700)
Line Height: 28px
Use: Card titles, section headers

H4 - Label
Font: 16px, Semi-Bold (600)
Line Height: 22px
Use: Form labels, list headers
```

#### Body Text

```
Body Large
Font: 16px, Regular (400)
Line Height: 24px
Use: Primary content, task descriptions

Body Regular
Font: 14px, Regular (400)
Line Height: 20px
Use: Secondary content, time labels

Caption
Font: 12px, Regular (400)
Line Height: 16px
Use: Metadata, helper text

Micro
Font: 11px, Medium (500)
Line Height: 14px
Use: Tiny labels, slider numbers
```

#### Special Use

```
Slider Value Display
Font: 32px, Bold (700)
Line Height: 40px
Letter Spacing: 1px
Use: Daily rating value

Button Text
Font: 16px, Semi-Bold (600)
Letter Spacing: 0.5px
Use: Primary CTAs
```

### Typography Best Practices

- **Hierarchy**: Always maintain clear visual hierarchy
- **Contrast**: Ensure 4.5:1 contrast ratio minimum (WCAG AA)
- **Line Length**: Max 70 characters for readability
- **Alignment**: Left-align body text, center titles when appropriate
- **Spacing**: Use consistent vertical rhythm (8px base unit)
